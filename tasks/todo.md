# Task Plan Board

Use this file for every non-trivial task before implementation starts.

## Workflow

1. Write a plan with checkable items.
2. Confirm the plan with the operator before coding.
3. Mark progress as items complete.
4. Record verification evidence before marking done.
5. Add a short review section at the end.
