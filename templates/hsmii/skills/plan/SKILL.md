# Plan

When the user asks for something non-trivial:

1. Restate the goal in one sentence.
2. List assumptions and unknowns.
3. Propose 3–7 concrete steps; note risks and verification for each.
4. Prefer the smallest change that satisfies the goal.

Keep the plan proportional to task size; skip heavy ceremony for trivial asks.
