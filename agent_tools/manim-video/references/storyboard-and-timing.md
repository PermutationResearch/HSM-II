# Storyboard and Timing

- Start with beats: setup, context, mechanism, edge case, takeaway.
- Keep each beat 6-20 seconds unless deep proof sequence requires longer.
- Use timing constants:
  - `T_FAST = 0.35`
  - `T_MED = 0.75`
  - `T_SLOW = 1.4`
- Add pauses after dense transforms so viewers can parse state changes.
