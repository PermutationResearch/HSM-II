# Composition Patterns

- **Field + subject**: use a low-contrast procedural background and one dominant focal object.
- **Asymmetric balance**: keep visual weight off-center for more dynamic scenes.
- **Rhythm bands**: organize the canvas into 2-4 horizontal/vertical rhythm zones.
- **Foreground accents**: add sparse, high-contrast accents only after base layering.
- **Negative space**: leave intentional quiet regions for readability and depth.
